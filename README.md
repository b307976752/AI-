# AI-中医旺旺服务协议页面

## 项目说明
这是AI-中医旺旺应用的用户服务协议和支付页面。

### 主要功能
- 用户服务协议展示
- 用户协议确认机制
- 支持月度和年度会员选择
- 支持微信和支付宝支付方式
